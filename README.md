# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project demonstrates basic game logic, DOM manipulation, and a clean user interface.

---

## 🚀 Features

- 🧑‍🤝‍🧑 Two-player turn-based gameplay (X & O)
- 🏆 Automatic winner detection
- 🔄 Reset and New Game functionality
- 🎨 Modern and responsive UI design
- ⚡ Lightweight and fast (no external libraries)
